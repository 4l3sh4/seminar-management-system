PROJECT TITLE: Research Seminal System
PURPOSE OF PROJECT:The Research Seminar System is a Java-based desktop application that manages research seminar activities for a university/faculty environment. It supports multiple user roles (Student, Evaluator, Coordinator) to streamline user registration/login and seminar-related workflows. The system aims to centralize seminar information, reduce manual paperwork, and provide an organized way to manage submissions, evaluations, and coordination tasks within a single department.
